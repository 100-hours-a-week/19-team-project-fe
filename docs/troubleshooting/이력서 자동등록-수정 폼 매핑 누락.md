# 이력서 자동등록/수정 폼 매핑 누락

## 증상

- 이력서 자동 등록 후 경력(careers) 데이터가 폼에 채워지지 않음
- 이력서 상세/수정에서도 경력이 빈 값으로 보임

## 발생 위치

- `src/widgets/resume-edit/ui/ResumeEditPage.tsx`
- `src/entities/resumes/api/parseResumeSync.ts`

## 원인

- `content_json.careers`를 문자열 배열만 가정하고 파싱
- 실제 응답은 객체 배열(`company/job/position/start_date/end_date`) 형태로 내려옴

## 해결

- `careers`를 문자열/객체 모두 처리하도록 정규화 로직 추가
- 날짜 기반 기간 문자열을 생성해 폼에 표시
- 파싱 타입에 `ResumeParseCareer` 추가해 객체 형태 허용

## 적용한 수정 요약

- `ResumeParseContentJson.careers`를 `string | ResumeParseCareer`로 확장
- `normalizeCareerItems`로 폼 매핑 일원화
