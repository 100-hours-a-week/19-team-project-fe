# BFF 집계 적용 실행 계획

## 1. 단계별 적용 순서
1) /me/edit 집계 적용
2) /onboarding/profile 집계 적용
3) /chat 집계 적용
4) /experts/[id] 집계 적용
5) /chat/[chatId] 집계 적용

## 2. 각 단계 체크리스트
- BFF 엔드포인트 설계 및 구현
- 기존 API 호출 제거 및 단일 호출로 대체
- 에러 처리/로딩 상태 단순화
- 통합 테스트 및 회귀 확인

## 3. 리스크 및 대응
- BFF 장애 시 영향 확대 → fallback 전략 필요
- 응답 스키마 변경 시 FE 영향 → 버전 관리 필요
- 인증 처리 누락 위험 → 토큰/쿠키 처리 표준화

## 4. 테스트 플랜
- 핵심 화면 E2E 테스트
- API 계약 테스트(요청/응답 스키마 검증)
- 네트워크 요청 수 비교(적용 전/후)
